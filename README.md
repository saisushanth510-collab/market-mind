# MarketAI Suite - AI Powered Sales & Marketing Assistant

A lightweight, AI-powered marketing and sales assistant built with Flask and Groq API using the LLaMA 3.3 70B model.

## Features

- **Campaign Generator**: Create comprehensive marketing campaigns with AI
- **Sales Pitch Generator**: Craft compelling sales pitches tailored to your audience
- **Lead Scoring Tool**: Automatically score and prioritize leads
- **Clean, Modern UI**: Responsive design that works on all devices
- **Fast AI Processing**: Powered by Groq's high-speed inference

## Prerequisites

- Python 3.8 or higher
- Groq API key (get it from [console.groq.com](https://console.groq.com))
